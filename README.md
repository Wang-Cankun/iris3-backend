# iris3-backend

## Description

This is the IRIS3 backend server, including:


- Graphql
- Based on [Prisma starter kit](https://github.com/fivethree-team/nestjs-prisma-starter)
- Unit test and e2e test
- GitHub actions and Codecov setup

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
